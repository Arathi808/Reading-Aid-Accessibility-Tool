# Reading Aid for Individuals with Learning Disabilities Using Text-to-Speech Technology

## Overview

This project is a web-based accessibility platform designed to assist individuals with learning disabilities and provide a basic hearing assessment tool. The system combines Text-to-Speech (TTS) technology with an interactive hearing test, making reading and accessibility support available through a simple and user-friendly interface.

## Features

### Text-to-Speech Module

* Converts text into spoken audio.
* Supports English and Hindi languages.
* Allows users to enter custom text.
* Supports PDF document upload and reading.
* Stop and resume speech functionality.

### Hearing Assessment Module

* Interactive hearing test using multiple audio frequencies.
* Frequency range testing at:

  * 250 Hz
  * 500 Hz
  * 1000 Hz
  * 2000 Hz
  * 4000 Hz
* User response-based evaluation.
* Provides preliminary hearing assessment results.

### User Interface

* Responsive design for desktop and mobile devices.
* Accessibility-focused layout.
* Simple navigation system.
* Modern visual design with interactive animations.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Web Speech API
* Web Audio API
* PDF.js

## Project Objective

The objective of this project is to improve accessibility and support individuals with learning disabilities by providing an easy-to-use reading aid through Text-to-Speech technology while also offering a preliminary hearing assessment tool.

## Future Enhancements

* Additional language support.
* Voice selection options.
* Speech speed customization.
* User authentication.
* Result history storage.
* Advanced hearing assessment analytics.
